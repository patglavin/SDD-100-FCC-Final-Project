How this page was assembled

This page was created using HTML and CSS. I tried to keep structural and stylistic CSS separate, resulting in my "body.css" and "coloring.css" files. Images and video had to be tweaked often to fit within my design goal, so they went into a separate file as well, to make jumping between "imagesAndVideo.css" and my HTML pages simpler.

This final HTML page has a separate CSS file, where I was able to take what I'd worked with and liked about my previous pages, and distil them down into one CSS file with new coloring. Given more time to experiment, this could be cut down to 1 or 2 files with something like Sass's variables and translation.

This project also utilized Bootstrap, with only a few minor tweaks to the navbar and the styling of my jumbotrons, removing the rounded corners of the Bootstrap 4 Beta version I'm using, bringing them closer to Bootstrap 3's squared design.

Finally, this was my first project completed with VS Code, rather than Sublime. The add ons are fantastic to work with, reminding me of NodeJS packages, where trying to customize Sublime leads to the headache of navigating a massive JSON file.
